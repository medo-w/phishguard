# 🎣 PHISHGUARD: Phishing URL Detection Tool
### Powered by Machine Learning (XGBoost Algorithm)

## 📌 Overview
[cite_start]**PHISHGUARD** is a security solution developed to identify and mitigate phishing URLs using machine learning techniques[cite: 52, 256]. [cite_start]By analyzing URL features, the system can distinguish between legitimate and malicious links with high precision, providing an essential layer of defense against social engineering attacks[cite: 55, 258].

[cite_start]This project was developed as part of my specialization in **Information Security** at **Universiti Tun Hussein Onn Malaysia (UTHM)**[cite: 7, 257].
